## 🔑 关键提示词规则

### 面部保留规则（所有主题通用）
```
🚨 CRITICAL FACE PRESERVATION RULES (ABSOLUTE PRIORITY - NON-NEGOTIABLE):

‼️ YOU MUST USE THE EXACT FACE FROM THE INPUT IMAGE - DO NOT GENERATE A NEW FACE ‼️

1. 输入图像中的面部必须像素级别精确转移到输出
2. 不要创建、生成、想象或幻想新面部 - 使用实际捕获的面部
3. 禁止：任何 AI 生成的面部、合成面部或不是来自输入图像的面部
4. 精确保留：面部结构、眼睛、鼻子、嘴巴、下巴、颌线、颧骨、前额
5. 精确保留：肤色、皮肤纹理、皱纹、年龄线、雀斑、痣、疤痕、胡须
6. 不要修改、增强、平滑、美化、衰老或以任何方式改变面部
7. 只有照明和色彩分级可以调整 - 面部特征保持 100% 相同
8. 该人必须立即被识别为输入照片中的同一个人
```

### 输出格式（所有主题通用）
```
CRITICAL OUTPUT FORMAT:
- 完全忽略参考图像的宽高比
- 输出必须是：竖向 1080x1920 像素（垂直/高格式）
- 不要创建方形图像 - 输出必须是竖向肖像
```
